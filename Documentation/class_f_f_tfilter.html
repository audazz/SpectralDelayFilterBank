Detailed Description

FFTfilter class for preforming filtering on signals.

This class uses the overlap-add method to preform convolution on a real-valued signal. It requires the fftw library to work properly.

This method preforms the filtering (or more generally convolution) of the signal passed as input via the overlap-add algorithm. The output buffer gets overwritten during the process. It is possible to use the input buffer as the output buffer (the overwriting still applies).
